# Фаза 7: UX Polish + Developer Experience

> **Статус:** ⏳ Ожидает
> **Длительность:** 1.5-2 недели

Цель: улучшение пользовательского опыта и инструментов разработчика.

---

## 1. Command Palette (⌘K) 🆕

- Быстрый доступ ко всем функциям: создание задачи, переключение personality, поиск по памяти/KB, запуск workflow
- Hotkey: Cmd+K / Ctrl+K
- Команды с autocomplete, недавние действия, fuzzy search
- **Расширяемая система команд**: skills могут регистрировать свои команды в палитре
- **Slash-команды**: прямо в поле ввода `/memory`, `/wiki`, `/task`, `/research`

## 2. Split View / Canvas Mode 🆕

- Рабочее пространство рядом с чатом: слева диалог, справа код/документ/результат
- **Режимы:**
  - **Research canvas**: слева чат, справа отчёт с цитатами, browser screenshots
  - **Code canvas**: слева чат, справа редактор кода с файловым деревом, diff preview
  - **D&D canvas**: слева чат, справа campaign state, карта, NPC sheet
  - **Data canvas**: слева чат, справа таблица данных и графики
- Ресайзинг панелей, сохранение layout'а
- **Artifact preview**: клик на artifact в чате открывает его в правой панели

## 3. Prompt Playground 🆕

- Отдельный UI для тестирования и отладки промптов
- **A/B тестирование**: сравнительный анализ двух версий системного промпта на одном запросе
- **Side-by-side сравнение моделей**: одинаковый запрос к разным провайдерам/моделям
- **Переменные в промптах**: `{{user_name}}`, `{{context}}`, `{{tools}}` — подстановка значений
- **История версий**: каждое изменение промпта сохраняется, можно вернуться к любой версии
- **Token counter**: live-подсчёт токенов промпта
- **Export промпта**: скопировать в формате для OpenAI/Anthropic/local
- **Quick test**: отправить промпт в чат прямо из Playground

## 4. Conversation Templates 🆕

- Сохранённые шаблоны запросов с placeholders: `"Напиши тесты для {module}"`, `"Проанализируй {url}"`
- Поддержка переменных, которые подставляются перед отправкой
- Категории шаблонов, pinned templates
- **Quick actions**: шаблон может быть привязан к кнопке в интерфейсе

## 5. Onboarding & Help System 🆕

- Интерактивный онбординг при первом запуске (walkthrough)
- Контекстная помощь: "?" рядом с каждым элементом UI
- Встроенный FAQ и документация по возможностям
- **Примеры запросов**: галерея промптов для разных сценариев

## Деливерабл

Command Palette для быстрой навигации, Split View для продуктивной работы, Prompt Playground для разработки промптов, система шаблонов и онбординг.
